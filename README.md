# Sales Dashboard Analysis (Power BI)

## Overview
This project presents an interactive sales dashboard built using Power BI to analyze business performance across regions, products, and customer segments.

## Objectives
- Analyze revenue, profit, and order trends
- Identify top-performing products and regions
- Evaluate customer segmentation
- Assess profitability and discount impact

## KPIs
- Total Revenue
- Total Profit
- Total Orders
- Average Order Value
- Profit Margin

## Dashboard Features
- Time series analysis of sales
- Revenue by city
- Sales by customer segment
- Top-performing products and sub-categories
- Profit vs Sales relationship (scatter plot)
- Interactive filters (date, region, category)

## Key Insights
- West region dominates total revenue
- Consumer segment generates the highest sales
- Technology category has the highest profitability
- Some products show high sales but low profit as furniture

## Tools Used
- Power BI (data visualization)
- Power Query (data transformation)
